## SQL Solutions

### Hotel Management System
- Designed relational schema for:
  - Users
  - Bookings
  - Items
  - Booking Commercials  

### Key Queries Implemented
- Last booked room for each user  
- Total billing for bookings (monthly)  
- Bills greater than a threshold value  
- Most & least ordered items per month  
- Second highest bill per month  

---

### Clinic Management System
- Designed schema for:
  - Clinics
  - Customers
  - Sales
  - Expenses  

### Key Queries Implemented
- Revenue by sales channel  
- Top 10 valuable customers  
- Monthly revenue, expense, and profit analysis  
- Most profitable clinic per city  
- Second least profitable clinic per state

## Python Solutions

### Time Converter
- Converts minutes into readable format (hours + minutes)  
- Example:
  - 130 → 2 hrs 10 minutes  

---

### Remove Duplicates
- Removes duplicate characters from a string  
- Maintains order using loop  
- Example:
